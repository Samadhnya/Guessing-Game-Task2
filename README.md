Guess the Number Game (Java)
Overview

This is a simple Java-based number guessing game. The program generates a random number between 1 and 100, and the player has to guess the correct number. After each guess, the program provides a hint indicating whether the guessed number is greater than or smaller than the secret number. The game continues until the correct number is guessed.

Features
Generates a random number between 1 and 100.
Accepts user input through the keyboard.
Validates user input to prevent invalid entries.
Provides hints after each guess.
Tracks the number of attempts made by the player.
Awards points based on the number of attempts taken to guess the correct number.
User-friendly console interface.
How the Game Works
The program generates a random number between 1 and 100.
The user enters a guess.
If the guess is greater than the secret number, the program displays "Greater".
If the guess is smaller than the secret number, the program displays "Smaller".
If the guess is correct, the program displays "Right" along with the total attempts and points earned.
The game ends when the correct number is guessed.
Scoring System
1st Attempt: 10 Points
2nd Attempt: 9 Points
3rd Attempt: 8 Points
Continues decreasing with each attempt.
Minimum score awarded is 1 point.
Requirements
Java Development Kit (JDK 8 or above)
Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or Command Prompt
How to Run
Save the file as task2.java.
Open Command Prompt or Terminal.

Compile the program:

javac task2.java

Run the program:

java task2
Start guessing the number and enjoy the game.
